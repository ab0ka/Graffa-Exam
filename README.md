# Graffa Exam Planner

Интерактивный статический прототип для планирования подготовки к экзаменам.

## Страницы

- `index.html` — первая экспериментальная версия с живым графом-капельками.
- `academic.html` — основная версия для учителя: темы, дни, материалы, календарь и план подготовки.

## Возможности

- Карта тем и учебных дней.
- Календарь подготовки.
- Загрузка материалов с подписью и описанием.
- Связи между темами, днями, материалами, заданиями и контрольными точками.
- Перетаскивание карточек, масштабирование, удаление связей и элементов.
- Сохранение состояния в `localStorage` браузера.

Проект не требует сборки: достаточно открыть `academic.html` или `index.html` в браузере.
